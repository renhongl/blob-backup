原型模式

类并不是必须的，对象未必需要从类中创建而来，一个对象是通过克隆另外一个对象所得到的。

原型模式不但是一种设计模式，也被称为一种编程泛型。

ECMAScript5提供了Object.create方法，可以用来克隆对象。

但是create方法性能不如 var obj = {} 或者 var obj = new Object();

以上两种替代方式，内部都是克隆原型而得到对象。
